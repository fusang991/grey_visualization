# 格雷码 × 超立方体 — 交互式可视化

在线体验：https://fusang991.github.io/grey_visualization/

## 这是什么

一个单文件、零依赖的交互式网站，用可视化的方式理解格雷码的数学本质：

> **格雷码 = n 维超立方体上的哈密顿环**

## 包含内容

### 可视化部分

- **二维** — 正方形上的行走，对比格雷码（沿棱走）vs 二进制（穿越对角线）
- **三维** — Three.js 渲染的可旋转立方体，动画演示哈密顿环
- **镜像构造** — 5 步递归动画，展示为什么"反转"能拼成环
- **交互探索** — 滑块选 n=2~6，查看完整格雷码序列 + 超立方体投影

### 挑战关卡（三维哈密顿环实验室）

1. **补全环** — 系统预选 2 条边，你补全哈密顿环
2. **受限环** — 禁止边约束下的哈密顿环
3. **有解还是无解** — 判断是否存在环，选择正确的数学理由
4. **最优环（TSP）** — 每条边带权，寻找最小代价哈密顿环
5. **规律发现** — 通关后揭示维度翻转的不变量 (4,2,2)

直接浏览器打开 `index.html`，无需服务器、无需构建。
